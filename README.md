# Hybrid Recommender System

## Business Problem
An e-commerce platform aims to increase user engagement by recommending relevant products to users.

## Methods Used
- User-Based Collaborative Filtering
- Item-Based Collaborative Filtering
- Content-Based Filtering
- Hybrid Recommendation Strategy

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn

## Output
The system generates Top-N product recommendations for a given user.
